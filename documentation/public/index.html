Zero Dependencies

Built entirely on Node.js core — no node_modules needed. ORM database drivers are the only optional extras.

ORM & Database

LINQ-style query builder, schema DDL, and 7 adapters — SQLite, MySQL, PostgreSQL, MongoDB, Redis, Memory, and JSON.

Modular Router

Sub-apps, groups, chaining, param handlers, and route introspection.

Security Suite

Helmet headers, CSRF protection, signed cookies, and timeout guards.

Body Parsers

Built-in json(), urlencoded(), text(), raw(), and multipart().

CORS

Cross-origin resource sharing with preflight and origin control.

HTTP Client

Built-in server-side fetch with proxy and TLS passthrough.

Rate Limiting

Per-IP request throttling with configurable windows and limits.

Compression

Brotli, gzip, and deflate with threshold and content filtering.

Static Files

Content-Type detection, ETag caching, and index file support.

Streaming Uploads

Multipart file streaming to disk with size limits and progress.

Validation

Schema-based request validation with type coercion and custom rules.

HTTPS & TLS

Native HTTPS with req.secure and secure-only routing.

Error Handling

Typed HTTP and framework error classes with global error handling.

Real-time

WebSocket rooms, pooling, broadcasting. SSE with auto-IDs and keep-alive.

Logging & Debugging

Request logger with timing, request IDs, and debug output.

Environment Config

Typed .env loading with schema validation and default values.
